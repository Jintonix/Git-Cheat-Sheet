\section{Git config}
To set a username use: \gitinline{git config --global user.name "YourName"}
\newline\noindent To set an e-mail use: \gitinline{git config --global user.email "YourE@mail"}
\newline\noindent This sets things up globablly. If you want to set things up for a specific repository just ommit the \gitinline{--gloabl} flag.
